# Equation Solver 🧮

A Python application that solves linear and quadratic equations.

## Features
- Solve linear equations (ax + b = 0)
- Solve quadratic equations (ax² + bx + c = 0)
- Clean modular code structure

## Tech Stack
- Python

## How to Run
1. Clone the repository
2. Run: `python main.py`

## Project Structure
- `main.py` - Main entry point
- `linear_equation.py` - Linear equation solver
- `quadratic_equation.py` - Quadratic equation solver
- `utils.py` - Utility functions
